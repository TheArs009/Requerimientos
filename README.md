# RequeriminetosGit
# Sistema de vacantes

# requerimientos funcionales: 

# - el sistema debe permitir que el administrador publique las vacantes disponibles
# - El sistema debe de permitir el registro de los usuarios a las vacantes disponibles
# - El sistema debe de informar al usuario sobre el estado de su registro
# - El sistema debe permitir que el administrador acepte o rechace las postulaciones
# - El sistema debe permitir la busque de vacantes de su interes 

# No funcionales:
# - El sistema debe de ser accesible desde cualquier dispositivo 
# - El sistema debe de mantener la seguridad de los datos de los usuarios las 24 horas del dia
# - El sistema debe permitir la comunicacion del usuario y el administrador 
# - El sistema debe de tener la capacidad de mantener la conexion de multiples usuarios 
# - El sistema debe permitir una respuesta rapida de menos de un segundo
